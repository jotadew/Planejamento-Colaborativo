Tela de Histórico de Consultas

Essa tela vai mostrar todas as interações anteriores do usuário com a IA, incluindo perguntas e fotos enviadas, bem como as respostas recebidas.

Entre os elementos da tela estão:

lista de consultas anteriores com data e hora
visualização de perguntas enviadas
visualização de respostas recebidas
opção de filtrar ou buscar consultas específicas
botão para excluir consultas individuais
indicação de status das consultas (processada, pendente, etc.)

A interface deve ser organizada e permitir que o usuário encontre facilmente informações de consultas passadas.
