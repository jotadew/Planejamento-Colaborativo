Filtros de Daltonismo

Esta funcionalidade permite que usuários com daltonismo consigam diferenciar cores na interface do sistema, melhorando a acessibilidade.

Entre os filtros disponíveis estão:

ajuste de cores para diferentes tipos de daltonismo (protanopia, deuteranopia, tritanopia)
destaque automático de elementos importantes
alterações na paleta de cores para melhor contraste
verificação de contraste antes de exibir novos elementos

O objetivo é garantir que todos os usuários consigam utilizar o sistema sem dificuldades, independentemente do tipo de daltonismo.
