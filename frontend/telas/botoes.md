Botões do sistema

Este arquivo descreve os principais botões que serão utilizados na interface do sistema.

Os botões são responsáveis por permitir a interação do usuário com a aplicação, sendo essenciais para navegação e execução das ações principais.

Entre os botões definidos estão:

botão de cadastro, utilizado para criar uma nova conta
botão de login, utilizado para acessar o sistema
botão de envio de imagem, utilizado para enviar fotos de sementes, pragas ou doenças
botão de confirmação, utilizado para concluir ações
botão de cancelamento, utilizado para interromper ações
botões de navegação, utilizados para se mover entre as telas

Em relação à interface, os botões devem ser simples, visíveis e fáceis de entender. Também é importante que tenham um bom contraste e algum tipo de resposta visual quando forem clicados.

Esses elementos são fundamentais para garantir uma boa experiência de uso no sistema.
