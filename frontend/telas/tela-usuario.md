Tela da Conta/Usuário

Essa tela vai mostrar as informações do usuário logado e permitir que ele edite seus dados.

Entre os elementos dessa tela estão:

exibição do nome do usuário
exibição do e-mail
botão para editar informações
botão para alterar senha
botão para salvar alterações
botão para sair da conta

A interface deve ser clara e fácil de usar, com botões visíveis e organizados. Essa tela garante que o usuário possa gerenciar sua conta de forma simples.
