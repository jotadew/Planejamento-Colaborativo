Sistema Offline para áreas rurais

Essa funcionalidade permite que o usuário utilize o aplicativo mesmo em locais sem conexão com a internet, garantindo acesso a informações importantes e histórico de consultas.

Entre os recursos do sistema offline estão:

acesso ao histórico de consultas previamente realizadas
envio de perguntas e fotos que ficam armazenadas localmente até reconectar
sincronização automática quando houver conexão disponível
armazenamento seguro de dados do usuário
interface funcional mesmo sem internet

O objetivo é garantir que usuários em áreas rurais possam utilizar o sistema de forma contínua e confiável.
