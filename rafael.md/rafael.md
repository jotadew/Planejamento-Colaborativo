#Tela de Cadastro

Permite que novos usuários criem uma conta no sistema.

- campos para nome, e-mail e senha
- botão para confirmar cadastro
- validação básica de dados

#Botões do sistema

Define os botões principais da interface.

- botão de cadastro
- botão de login
- botão de envio de imagem
- botão de confirmação
- botão de cancelamento
- botões de navegação

#Tela da Conta/Usuário

Permite que o usuário veja e edite seus dados.

- exibição do nome e e-mail
- botão para editar informações
- botão para alterar senha
- botão para salvar alterações
- botão para sair da conta
